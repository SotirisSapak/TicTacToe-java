# TicTacToe
A simple TicTacToe console game written in java

## 🎮 Gameplay
🆔 Give the X player name... <br />
🆔 Give the O player name... <br />

🔢 Use your numpad numbers to choose the table position to insert your option <br />
✅ That's it! Enjoy
